# 🛍️ Cyber Next E-commerce  

A modern e-commerce website built with **Next.js** and **Tailwind CSS**.  

## 🚀 Features  
- Fast and responsive  
- Dark mode support  
- Authentication with Supabase  
- Shopping cart functionality  

## 🛠 Tech Stack  
- **Frontend:** Next.js, React, Tailwind CSS  
- **Backend:** Supabase
